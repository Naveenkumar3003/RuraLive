INTRODUCTION
Smart Bus Tracking System for Rural Areas

The Smart Bus Tracking System is a digital solution designed to improve transportation in rural regions where bus timings are unpredictable and information is limited. This project provides real-time bus location, estimated arrival time, and SMS alerts for passengers without smartphones. The goal is to reduce waiting time, improve safety, and make rural commuting more reliable and efficient.
